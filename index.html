About the Game

This is a fun match-three game where you try to get three identical symbols in a row. Tap to start rolling the icons and see if you can win!

How to Play

1. Tap the "Start" button to begin the game.

2. The icons will start rolling randomly.

3. Try to get three identical symbols in a row to win.

4. If you do not get three in a row, you lose and can try again by tapping "Start".
